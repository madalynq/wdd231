Crochet Abbreviations (U.S. Terms)

  • ch - chain
  • sc - single crochet
  • hdc - half double crochet
  • dc - double crochet
  • tr - treble crochet
  • dtr - double treble crochet
  • sl st - slip stitch
  • inc - increase (2 stitches in one)
  • dec - decrease (combine stitches)
  • tog - together (e.g., dc2tog)
  • flo - front loop only
  • blo - back loop only
  • sk - skip
  • rep - repeat
  • rnd - round
  • row - row
  • sp - space
  • st - stitch
  • RS - right side
  • WS - wrong side
  • approx. - approximately
  • gauge - stitch/row count per inch/cm
  • * to * - repeat from * to *
  • dc2tog - double crochet 2 stitches together
  • sc2tog - single crochet 2 stitches together
  • puff - puff stitch
  • bobble - bobble stitch
  • cluster (cl) - cluster stitch
